# Test Plan

## In Scope

### Exchange Core
- Registration
- KYC
- Deposit and funding
- Spot order entry
- Trade execution validation
- Security and 2FA

### Web3 Wallet
- Wallet setup
- Backup and recovery
- Asset and network visibility
- Send and receive
- Transaction signing
- dApp connection

### API and Data Validation
- wallet balance endpoints
- order and trade APIs
- transaction history
- fee validation
- duplicate and missing record checks
- backend reconciliation

## Out of Scope
- live market performance benchmarking
- penetration testing
- blockchain node-level validation
- real-money production transactions

## Entry Criteria
- test artifacts prepared
- environment assumptions documented
- required sample data available

## Exit Criteria
- critical flows executed
- high-severity issues reviewed
- major reconciliation checks completed
- release checklist updated
