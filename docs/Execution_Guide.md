# Execution Guide

## Recommended Review Order
1. Read the main README and architecture overview
2. Review combined strategy and risk matrix
3. Open exchange and wallet test suites
4. Review API and SQL validation assets
5. Check bug samples and release-readiness artifacts

## Suggested Execution Sequence

### Exchange Core
- account registration
- KYC validation
- wallet funding
- spot trading
- 2FA and account security

### Web3 Wallet
- wallet creation or import
- backup and recovery validation
- asset and network checks
- send and receive scenarios
- dApp connection and signing flows

### Data and Reconciliation
- compare expected vs actual balances
- verify transaction history accuracy
- check order totals, filled quantities, and fees
- look for duplicates and missing records

## Expected Outputs
- pass/fail tracking in workbooks
- defect notes or linked reports
- summary report update
- release recommendation based on residual risk
